# ycw3320.github.io

개인 개발 블로그. 웹 개발 이슈, 기술 학습, 환경 세팅 팁을 기록합니다.

**https://ycw3320.github.io/**

## 기술 스택

| 영역 | 기술 |
|------|------|
| 프레임워크 | [Astro](https://astro.build/) v5 |
| 템플릿 | [AstroPaper](https://github.com/satnaing/astro-paper) v5.5.1 |
| 스타일링 | [Tailwind CSS](https://tailwindcss.com/) v4 |
| 검색 | [Pagefind](https://pagefind.app/) |
| 배포 | GitHub Pages + GitHub Actions |

## 로컬 실행

```bash
npm install
npm run dev      # http://localhost:4321
```

## 게시글 작성

`src/data/blog/` 에 `.md` 파일 추가 후 `main` 브랜치에 푸시하면 자동 배포됩니다.
